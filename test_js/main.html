JavaScript - это язык сценариев для интернета. Он является интерпретируемым языком : для написания и запуска скриптов не требуется компилятора, как в других языках. Можно просто написать сценарий (в html документе или в отдельном js файле), затем загрузить страницу в web-браузере. Он обычно применяется в качестве встраиваемого инструмента для программного доступа к различным объектам приложений.

Для чего нужен JavaScript?

  • Математические операции
  • Обработка и валидация данных в HTML-формах
  • Взаимодействие с пользователем и событиями
  • Взаимодействие с HTML-элементами на странице и управление их содержимым и стилями
  • Добавление анимации и различных графических эффектов на веб-страницы
Минусы

Язык компилируется в момент исполнения кода. Из-за этого увеличивается время выполнения программы.
